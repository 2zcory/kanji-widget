# Kanji Widget v1.5.0

Release date: 2026-03-13

## Highlights

- Refreshed the main screen with a stronger study-first layout, clearer today summary, and one dominant continue-learning action
- Added a lightweight widget setup flow when placing a new widget, including per-widget background opacity presets
- Improved the recent-study and widget-control surfaces so the launcher feels more intentional without becoming a heavy dashboard

## Improvements

- New widgets can now keep their own background opacity instead of sharing one global appearance setting
- Existing widgets continue to render safely by falling back to the shared default opacity
- Updated the launcher copy and supporting actions so empty states and no-widget states guide the next step more clearly

## Notes

- Existing widget learning interactions such as reveal, next, resize behavior, and detail-screen navigation remain unchanged
- Older widgets already on the home screen continue working after the update without needing to be recreated
- The shared widget-opacity control in the app now acts as the default opacity for widgets that do not use a per-widget override
