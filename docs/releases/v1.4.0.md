# Kanji Widget v1.4.0

Release date: 2026-03-13

## Highlights

- Added English and Vietnamese interface support across the app and home screen widget
- Added an in-app language picker so the app can follow the system language or switch directly between English and Vietnamese
- Refreshed localized widget states, actions, and study-summary text so the widget flow stays clear in both supported languages

## Improvements

- Localized pluralized study-time and open-count text for the main screen, detail screen, and stats surfaces
- Updated widget rendering so language changes refresh both app screens and active widget instances more reliably
- Added localization design notes and glossary support to make future language expansion easier

## Notes

- This release keeps the existing production package name and local study data intact
- Current in-app language options are System default, English, and Vietnamese
- Kanji content fetched from external sources may still appear in its original source language
