# Kanji Widget v1.6.0

Release date: 2026-03-14

## Highlights

- Added app-wide theme selection with `System`, `Light`, `Dark`, and a dedicated `Glass` mode
- Applied the first shipped Glass visual system across the main screen, Kanji detail, widget configuration, and study stats surfaces
- Added theme-aware popup overlays so dialogs match the active app theme and feel consistent across in-app settings flows

## Improvements

- The app now restores the selected theme on cold start without flashing the wrong mode first
- Glass mode now uses stronger hero separation, clearer elevated surfaces, and lighter button styling after review-driven UI refinement
- Theme, language, and widget-help popups now share one dismissal pattern, including tap-outside-to-close behavior
