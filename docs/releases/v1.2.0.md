# Kanji Widget v1.2.0

Release date: 2026-03-12

## Highlights

- Added a lightweight compound-examples section to the Kanji Detail screen
- Each supported compound row now shows the written form, reading, meaning, and a short usage hint
- Kept the detail screen compact by showing only a small filtered set of practical compounds

## Improvements

- Added local caching for compound examples so repeat detail opens stay fast and resilient
- Filtered and ranked compound suggestions to prefer more readable, common study words
- Updated the Kanji Detail design docs and implementation checklist to match the shipped compounds behavior

## Notes

- Compound examples currently come from `kanjiapi.dev` word data
- The compounds section stays hidden when no suitable examples are available for the current kanji
- Existing stroke-order playback, today stats, and next-random behavior remain unchanged
