# Kanji Widget v1.5.1

Release date: 2026-03-14

## Highlights

- Refreshed the Kanji Detail screen with a clearer study-first hierarchy and a stronger `Next random` continuation flow
- Improved readings, compounds, and today-stats presentation so the detail screen feels cleaner without becoming heavier
- Fixed random detail openings so the next kanji now loads its meaning reliably even when its cached detail was missing

## Improvements

- The Kanji hero, readings, and action areas now read more clearly as one focused study surface
- Compound rows are easier to scan when several examples are visible
- Today stats now support the detail screen without competing with the main study content
- Opening a random kanji from the detail screen or the home screen now fills missing detail data before showing the next entry

## Included Work

- PR `#9` refresh-kanji-detail-hierarchy
- Commit `944996e` load missing detail data for random kanji

## Notes

- Stroke-order playback, compound playback, today tracking, and next-random navigation remain part of the same Kanji Detail flow
- This release does not add a new data provider, a second detail mode, or account-backed study state
