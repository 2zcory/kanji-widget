# Kanji Widget v1.3.0

Release date: 2026-03-12

## Highlights

- Added on-device pronunciation playback to the Kanji Detail screen using Android `TextToSpeech`
- Added audio playback for the main kanji reading and eligible compound-example rows
- Switched the playback controls to compact audio icons so the detail screen stays lightweight

## Improvements

- Kept audio playback local-only without adding a remote audio source
- Added stable single-playback behavior so starting a new pronunciation stops the current one first
- Preserved the existing detail flow for stroke order, compounds, today stats, and next-random navigation

## Notes

- Pronunciation playback depends on Japanese `TextToSpeech` support being available on the device
- Audio controls stay unavailable when a usable reading or Japanese TTS support is not available
- Existing compound examples and study history remain stored locally on-device
