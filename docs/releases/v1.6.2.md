# Kanji Widget v1.6.2

Release date: 2026-03-14

## Highlights

- Fixed theme regressions introduced after the settings-screen release
- Restored legible Kanji stroke rendering in dark mode on the detail screen
- Corrected settings and detail-return behavior so launcher surfaces refresh cleanly

## Improvements

- `Light` now stays flat without unintended button or surface shadows outside `Glass`
- Returning from `KanjiDetail` now refreshes the main screen so recent rows no longer keep stale appearance state
- Recent-history cards now render with a dedicated dark-mode background path instead of falling back to the wrong surface

## Included Work

- PR `#13` fix theme and settings regressions

## Notes

- This is a focused hotfix release for regressions from `v1.6.1`; it does not change the settings-screen structure or add new feature scope
