# Kanji Widget v1.5.2

Release date: 2026-03-14

## Highlights

- Added Vietnamese kanji meanings when the app language is set to Vietnamese
- Extended Vietnamese meaning localization to compound rows on the Kanji Detail screen
- Preserved the original source meaning so English mode still renders the expected source-language gloss

## Improvements

- Meaning rendering now resolves through one locale-aware path across the widget, Kanji Detail, home summaries, and ranking rows
- Cached kanji entries can backfill Vietnamese meaning data lazily instead of pretending the source meaning is already Vietnamese
- Vietnamese meaning phrasing now translates normalized whole meaning phrases, which reads more naturally than fragment-by-fragment translation
