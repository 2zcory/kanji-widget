# Kanji Widget (Android)

Widget giúp người học Kanji xem nhanh 1 chữ mỗi lần cập nhật:
- Kanji
- Onyomi / Kunyomi
- Nghĩa tiếng Việt
- Ví dụ ngắn

## Tính năng
- Home Screen App Widget (4x2)
- Nút **Đổi chữ** để random Kanji mới
- Danh sách Kanji N5 mẫu (có thể mở rộng)

## Build
1. Mở project bằng Android Studio (JDK 17)
2. Sync Gradle
3. Run app để cài
4. Ra Home Screen > Widgets > kéo **Kanji học nhanh** ra màn hình

## Mở rộng
- Thêm dữ liệu N4/N3 từ JSON
- Thêm chế độ theo chủ đề (thời gian, thiên nhiên, động từ...)
- Lưu lịch sử chữ đã học
